Assignment 2: Ethereum basics
In this assignment you get familiar with the basics of Ethereum. You will learn how to connect to the Ethereum main network, access test networks, manage your accounts and execute transactions. You will also get started with Ethereum JavaScript API (web3.js) and do some basic operations in the test networks.
